Програма Літак
Написати клас(класи) для роботи із наборами рейсів літаків.
Інформація про рейс містить наступні поля:
•	Номер рейсу. Наприклад: KL 4103.
•	Авіакомпанія. Наприклад: British Airways.
•	Тип літака: виробник та модель. Перелік варіантів можливих значень:
	Виробник Airbus. Моделі: A310, A320, A330.
	Виробник Boeing. Моделі: 737, 747, 767, 777.
	Виробник Embraer. Моделі: 175, 190, 195.
•	Вартість квитка.
•	Відстань.


Треба мати можливість:
-	виведення змісту набору на екран відсортованому по: вартості квітка, або по відстані, або по авіакомпанії та вартості квітка, або по відстані, виробнику та моделі.
-	додавання рейсу;
-	пошук даного/даних по номеру рейсу, або по авівкомпаніі, або по вартості;
-	видалення даного/даних по номеру рейсу або по типу літака;

Крим цього, якщо є два набори рейсів L1 та L2, требо мати можливисть:
-	визначити, чи входить набір L1 в набір L2.
-	перевірити на рівність два набори L1 і L2.
-	додати в кінець набору L1 всі елементи набору L2.
-	сформувати набір L, включивши в нього по одному разу елементи, які входять хоча б в один з наборів L1 і L2.
-	об'єднати два упорядкованих набори L1 і L2 в один впорядкований набір L (впорядкування за Вашим бажанням).
-	сформувати набір L, включивши в нього по одному разу елементи, які входять одночасно в обидва набори L1 і L2.
